# creating a new file
```bash
touch dummy_file

open dummy_file

open -a atom dummy_file
```

# Deleting Files
```bash
rm -rf # deleting recursively

rm * # using a wildward to select everything

rm -r # deletes the child components of a folder
```

# Working Directory
```bash
pwd
```
